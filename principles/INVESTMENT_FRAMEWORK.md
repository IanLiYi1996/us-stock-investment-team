## 一、投资哲学

### 1.1 核心理念
- **买增长**：优先选择营收 / 利润高速增长的公司
- **买护城河**：技术壁垒、网络效应、转换成本高的标的
- **买趋势**：顺势交易，不抄底摸顶

### 1.2 不做的事
- ❌ 不炒概念、不追热点（无基本面支撑）
- ❌ 不重仓单一行业（集中度 < 40%）
- ❌ 不持有亏损扩大、现金流为负的公司
- ❌ 不在不确定事件前重仓（财报、FDA、FOMC）

## 二、标的筛选标准

### 2.1 进入条件（Entry Criteria）
| 维度 | 标准 | 权重 |
|------|------|------|
| 营收增速 | YoY ≥ 15% | 高 |
| 利润率趋势 | 毛利率 > 40%，且改善中 | 高 |
| 估值 | PEG < 1.5 或 PS < 行业均值 | 中 |
| 技术面 | 价格 > 200日均线，RSI 40-70 | 中 |
| 分析师评级 | 多数分析师 Buy/Strong Buy | 低 |
| 催化剂 | 有明确的近期催化剂事件 | 高 |

### 2.2 退出条件（Exit Criteria）
| 情况 | 操作 |
|------|------|
| 基本面逻辑被证伪 | 立即清仓，不等止损 |
| 营收 miss 连续 2 季 | 减仓 50% |
| 盈利 ≥ 8% / 15% / 25% | 分级止盈（见止盈规则） |
| 亏损 ≥ 12% / 18% | 分级止损 |
| 大盘 VIX > 35 | 整体仓位降至 50% 以下 |

## 三、仓位管理

### 3.1 仓位上限
- 单标的：≤ 20% 总资产
- 单行业：≤ 40% 总资产
- 现金留存：≥ 10%（用于抄底机会）

### 3.2 建仓策略
- **分批建仓**：目标仓位分 3 批建立（1/3 → 等回调 → 确认后补）
- **避免一把梭**：禁止单次建满目标仓位
- **先小后大**：先建 1/3 试仓，确认趋势再加仓

## 四、风险管理

### 4.1 组合层面
- 持仓数量：5-8 只（过少=集中风险，过多=难以跟踪）
- 行业分散：科技 / 消费 / 医疗 / 能源至少 2 个板块
- Beta 控制：组合整体 Beta < 1.3

### 4.2 单标的层面
- 止损：入场后立即设止损（-12% 减半，-18% 清仓）
- 止盈：纪律性止盈，不贪婪（+8% 减一成，+15% 再减，+25% 仅留尾仓）
- 持仓时间：预期 3-12 个月，不是隔夜炒短线

### 4.3 宏观风险
- FOMC 会议前后：减少新开仓
- VIX > 30：整体降仓至 60%，VIX > 40 降至 40%
- 大型地缘事件：等待 48h 观察，不盲目抄底

## 五、复盘机制

### 5.1 周度复盘
- 每周五收盘后：持仓盈亏统计
- 上周预判 vs 实际结果对比
- 下周重点关注事项

### 5.2 季度复盘
- 季度总收益 / 跑赢 / 跑输 S&P 500 对比
- 最成功的 3 笔交易 → 总结什么做对了
- 最失败的 3 笔交易 → 复盘哪里判断失误
- 策略规则是否需要更新
