# HEARTBEAT — CIO 自检清单

> 每日两次执行：盘前（美东 8:30）+ 收盘后（美东 16:30）

---

## 盘前自检

### 1. 持仓健康度
- [ ] 是否有仓位接近止损线（亏损 ≥ 10%）？
- [ ] 是否有仓位接近止盈线（盈利 ≥ 7%）？
- [ ] 隔夜是否有重大事件影响持仓？

### 2. 待办事项
- [ ] TASKS.md 是否有过期或阻塞的任务？
- [ ] 是否有未完成的 Research 派单？
- [ ] 昨日是否有未处理的 KO 通知？

### 3. 每日简报
- [ ] 准备 Top 3 关注标的 + 理由
- [ ] 昨日操作结果确认（仅已验证的结果）
- [ ] 今日关键催化剂事件

## 收盘后自检

### 1. 当日总结
- [ ] 今日执行了哪些交易？结果如何？
- [ ] 盘中是否触发了止损/止盈？
- [ ] 持仓整体盈亏情况

### 2. 知识沉淀
- [ ] 今日是否有 signal≥2 的 closeout 需要 KO 处理？
- [ ] 是否有新的 scar/pattern 需要记录？

### 3. 明日准备
- [ ] 明日有哪些标的的催化剂事件？
- [ ] 是否需要提前派单 Research？
